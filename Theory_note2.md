## ­Ъј» рЃЏрЃўрЃќрЃљрЃюрЃў  
рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃЏрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮрЃю:  
- рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃерЃў  
- рЃарЃЮрЃњрЃЮрЃа рЃўрЃЏрЃБрЃерЃљрЃЮрЃю рЃбрЃўрЃърЃћрЃЉрЃќрЃћ `int`, `float`, `bool`, `str`  
- рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў рЃЊрЃљ рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃћрЃЉрЃў  
- рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃЮрЃю рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃњрЃљрЃю (`input`)  
- рЃарЃЮрЃњрЃЮрЃа рЃўрЃЏрЃБрЃерЃљрЃЮрЃю рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃќрЃћ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃю `f-string`

## 1№ИЈРЃБ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў (Variables)

рЃфрЃЋрЃџрЃљрЃЊрЃў рЃљрЃарЃўрЃА рЃћрЃарЃЌрЃњрЃЋрЃљрЃарЃў рЃљрЃЊрЃњрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ.  

```python
name = "Nika"  
age = 15  
height = 1.75  
is_student = True
```
#### рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ:

`name`, `age`, `height`, `is_student` Рђћ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃљ

`=` Рђћ рЃЏрЃўрЃюрЃўрЃГрЃћрЃЉрЃўрЃА рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃфрЃЋрЃџрЃљрЃЊрЃА рЃљрЃюрЃўрЃГрЃћрЃЉрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА

Python-рЃерЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃбрЃўрЃърЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃ«рЃљрЃЊрЃћрЃЉрЃљ рЃљрЃа рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ Рђћ рЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃўрЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ


## 2№ИЈРЃБ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃў

РюЁ рЃАрЃгрЃЮрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў:
```
user_name = "Elene"
birth_year = 2009
is_ready = True
```
РЮї рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў:
```
2name = "Anna"     # рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃфрЃўрЃцрЃарЃўрЃЌ рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ
user-name = "Gio"  # рЃљрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ "-" рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
print = "Hi"       # рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ Python-рЃўрЃА рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃљ
```
рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃљ: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ `snake_case` рЃцрЃЮрЃарЃЏрЃљрЃбрЃў рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў.
рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: `student_name`, `total_price`, `user_age`


## 3№ИЈРЃБ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃбрЃўрЃърЃћрЃЉрЃў (Data Types)

| рЃбрЃўрЃърЃў | рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
|------|-----------|--------|
| `int` | 5, -12, 100 | рЃЏрЃЌрЃћрЃџрЃў рЃарЃўрЃфрЃ«рЃЋрЃў |
| `float` | 3.14, -0.5 | рЃљрЃЌрЃгрЃўрЃџрЃљрЃЊрЃў рЃарЃўрЃфрЃ«рЃЋрЃў |
| `bool` | True, False | рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ |
| `str` | "Hello", 'Python' | рЃбрЃћрЃЦрЃАрЃбрЃў (рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў) |

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ
```
x = 10  
y = 3.5  
z = "Python"  
is_online = False  
print(x, y, z, is_online)
```

| Variable | Type | Value |
|----------|------|-------|
| x | int | 10 |
| y | float | 3.5 |
| z | str | Python |
| is_online | bool | False |

## 4№ИЈРЃБ рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў (Arithmetic Operators)

| рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў | рЃерЃћрЃЊрЃћрЃњрЃў |
|------------|---------|-----------|---------|
| `+` | рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ | `5 + 3` | 8 |
| `-` | рЃњрЃљрЃЏрЃЮрЃЎрЃџрЃћрЃЉрЃљ | `10 - 4` | 6 |
| `*` | рЃњрЃљрЃЏрЃарЃљрЃЋрЃџрЃћрЃЉрЃљ | `3 * 5` | 15 |
| `/` | рЃњрЃљрЃДрЃЮрЃцрЃљ (float) | `7 / 2` | 3.5 |
| `//` | рЃЏрЃЌрЃћрЃџрЃў рЃњрЃљрЃДрЃЮрЃцрЃљ | `7 // 2` | 3 |
| `%` | рЃюрЃљрЃерЃЌрЃў | `7 % 2` | 1 |
| `**` | рЃ«рЃљрЃарЃўрЃАрЃ«рЃерЃў рЃљрЃДрЃЋрЃљрЃюрЃљ | `2 ** 3` | 8 |



рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
```
a = 7  
b = 2  
print(a + b, a - b, a * b, a / b, a // b, a % b, a ** b)
```
­ЪЊц рЃерЃћрЃЊрЃћрЃњрЃў: `9 5 14 3.5 3 1 49`


## 5№ИЈРЃБ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў (String Methods)

| рЃЏрЃћрЃЌрЃЮрЃЊрЃў | рЃљрЃдрЃгрЃћрЃарЃљ | рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў | рЃерЃћрЃЊрЃћрЃњрЃў |
|---------|---------|-----------|----------|
| `.upper()` | рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃљрЃАрЃЮ рЃЊрЃўрЃЊрЃљрЃЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ | `"python".upper()` | `"PYTHON"` |
| `.lower()` | рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃљрЃАрЃЮ рЃърЃљрЃбрЃљрЃарЃљрЃЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ | `"HELLO".lower()` | `"hello"` |
| `.title()` | рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃљрЃАрЃЮ рЃЊрЃўрЃЊрЃљрЃЊ | `"hello world".title()` | `"Hello World"` |
| `.replace()` | рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃА рЃљрЃю рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃЕрЃљрЃюрЃљрЃфрЃЋрЃџрЃћрЃЉрЃљ | `"banana".replace("a", "o")` | `"bonono"` |
| `.split()` | рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃЊрЃљрЃДрЃЮрЃцрЃљ рЃАрЃўрЃљрЃЊ | `"a b c".split()` | `["a", "b", "c"]` |
| `.join()` | рЃАрЃўрЃўрЃА рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃљрЃЊ рЃерЃћрЃгрЃћрЃЉрЃћрЃЉрЃљ | `"-".join(["a", "b"])` | `"a-b"` |
| `.strip()` | рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃфрЃўрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃў рЃЊрЃљ рЃЉрЃЮрЃџрЃЮрЃерЃў | `" hi ".strip()` | `"hi"` |
| `.count()` | рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ | `"banana".count("a")` | `3` |

## ­ЪДЕ 6№ИЈРЃБ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃћрЃЉрЃў (Strings)

рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃў рЃљрЃарЃўрЃА рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃўрЃА рЃЌрЃљрЃюрЃЏрЃўрЃЏрЃЊрЃћрЃЋрЃарЃЮрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃерЃў " " рЃљрЃю ' '.
```
text = "Python is fun!"
```

## ­Ъћа рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃљрЃфрЃўрЃљ (Indexing)

рЃўрЃюрЃЊрЃћрЃЦрЃАрЃљрЃфрЃўрЃљ рЃўрЃгрЃДрЃћрЃЉрЃљ 0-рЃЊрЃљрЃю.

```
word = "Python"
print(word[0])   # P
print(word[1])   # y
print(word[-1])  # n
```
## Рюѓ№ИЈ рЃАрЃбрЃарЃўрЃЦрЃЮрЃюрЃўрЃА рЃЊрЃљрЃГрЃарЃљ (Slicing)
```
text = "Programming"
print(text[0:6])  # 'Progra'
print(text[3:])   # 'gramming'
print(text[:5])   # 'Progr'
print(text[-4:])  # 'ming'
```
